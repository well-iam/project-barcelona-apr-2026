The Challenge

⚠ The Task

A robot navigating an industrial floor must classify every scene as STOP, SLOW, or CONTINUE. Wrong call = collision or gridlock.

📊 The Data

27 heterogeneous label categories from 4 source datasets. We unified them into 4 decision groups.

person hat helmet head → PERSON_LIKE

🎯 Scoring

Partial credit: exact match = 1.0, one level off = 0.3. Calibration and detection quality also scored.

🔍 No Test Annotations

Test set has zero labels. Object detection is mandatory — we must find everything ourselves.
